Class: Vector2

vector2~Vector2(a, b, isPolar)

Constructs a Vector2. A vector2 is a 2-dimensional vector.

Constructor

new Vector2(a, b, isPolar)

Parameters:
Name Type Default Description
a number 0 Corresponds to the x-component if isPolar=false. Otherwise corresponds to polar coordinate r.
b number 0 Corresponds to the y-component if isPolar=false. Otherwise corresponds to polar coordinate angle theta.
isPolar boolean false if the vector is expressed in polar coordinates. regardless of this flag's value, cartesian and polar coordinates will be calculated and stored. This flag is only for initialization.
