Class: DebugConsole

DebugConsole(target_element)

Represents a debug console in the page. Appends the console as a child of the specified target_element or the document body if none is provided.

Constructor

new DebugConsole(target_element)

Parameters:
Name Type Description
target_element Element an HTML element in which to construct the debug console. This should probably be a div or some similar element.
